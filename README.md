 # Sistema de Gestão de Projetos – PIT / Extensão
 Sistema desenvolvido para cadastro, controle e visualização de projetos acadêmicos.
 ## Tecnologias- Node.js + Express- SQLite- React + Vite
 ## Como rodar
 ### Backend
 cd backend
 npm install
 npm run init-db
 npm start
 ### Frontend
 cd frontend
 npm install
 npm run dev
 Acesse http://localhost:5173
